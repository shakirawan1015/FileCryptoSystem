# 🔐 FileCryptoSystem

A secure **file encryption and decryption system** that protects your sensitive files using AES-256 encryption.

## ✨ Features

- 🔒 Encrypt any file (text, images, PDFs, videos, etc.)
- 🔓 Decrypt encrypted files back to original
- 🔑 Password-based encryption with key derivation
- 🚀 Simple command-line interface
- 🛡️ Military-grade AES-256 encryption

## 📋 Requirements

- Python 3.8+
- Required packages (see `requirements.txt`)
